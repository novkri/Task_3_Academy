# TODO app


Для имитации базы данных использовался json-server

### Запуск с json-serverом (windows)
```
npm run serve

```
### Запуск с json-serverом (unix)
```
npm run serve2

```

### Запуск без сервера
```
npm run run

```
## Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
